# Projet fidélisation client

## Objectif
Construire un score de churn client pour identifier les clients à risque, analyser les facteurs explicatifs et proposer des actions marketing de rétention.

## Dataset
Telco Customer Churn (Kaggle)

## Stack
- Python
- SQL
- Scikit-learn
- XGBoost
- SHAP
- Power BI
